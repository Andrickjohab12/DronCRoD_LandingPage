<h2>Dron CRoD</h2>
<br><br>
Dashboard de monitoreo de Dron mediante Telémetria
<ul>
  <li>Batería</li>
  <li>GPS</li>
  <li>Cámara</li>
  <li>Reconocimiento Facial</li>
  <li>API weather</li>
</ul>
;:
  <br><br>
  Desarrolada con React,Next.js y Typescript
Para correr el servidor necesitarras los comandos:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Abre [http://localhost:3000](http://localhost:3000) con tu navegador para ver el resultado.
